# 🎉 Sticker Shortcuts - Chrome Extension

Type a shortcut in any chat editor and it pastes an image automatically!

## Available Shortcuts

| Type this + Space | Result |
|---|---|
| `:john` | Pastes john sticker |
| `:black-friday` | Pastes black-friday image |

## How to Install (takes 30 seconds)

1. **Unzip** the `sticker-extension.zip` file to any folder on your computer
2. Open **Google Chrome**
3. Go to `chrome://extensions/` in the address bar
4. Turn on **Developer mode** (toggle in the top-right corner)
5. Click **Load unpacked**
6. Select the **unzipped folder** (the one containing `manifest.json`)
7. Done! ✅

## How to Use

1. Open **Google Chat**, **Slack**, **Notion**, or any chat with a rich text editor
2. Type `:john` and then press **Space**
3. The shortcut text gets replaced with the image!

## Notes

- Chrome may show a warning about developer mode extensions — this is normal, just dismiss it
- After a Chrome update, you may need to re-enable the extension at `chrome://extensions/`
